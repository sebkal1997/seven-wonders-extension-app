# 7 wonders extension client
Frontend client which allows to:
- create room
- add members to the game
- transfer resources
- increase production of the resource
- change round

## Dependencies
- [Node.js](https://nodejs.org)
- [Npm](https://www.npmjs.com)
- [SocketIO](https://socket.io/)
## Building
```
npm install
npm run build
```
